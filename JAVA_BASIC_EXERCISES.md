# Java 基础练习题

这份练习清单放在项目根目录，方便你直接按顺序学习。

## 使用建议

- 按顺序完成，不要跳题。
- 每题自己新建一个类，先写 `main` 方法跑通，再考虑优化。
- 建议把练习代码放到 `src/main/java/practice/exercises` 包下。
- 可以直接复制 `src/main/java/practice/exercises/ExerciseTemplate.java` 作为起点。

## 第一阶段：语法与输出

1. 输出 `Hello, Java`。
2. 定义 `int`、`double`、`char`、`boolean` 四种变量并打印。
3. 计算两个整数的和、差、积、商、余数。
4. 交换两个变量的值，分别用“临时变量”和“不用临时变量”两种方式实现。
5. 将摄氏度转换为华氏度并输出结果。

## 第二阶段：输入与判断

6. 输入一个整数，判断它是奇数还是偶数。
7. 输入一个整数，判断它是正数、负数还是 `0`。
8. 输入三个整数，输出最大值。
9. 输入一个年份，判断是否为闰年。
10. 输入学生分数，输出等级：`90-100:A`，`80-89:B`，`70-79:C`，`60-69:D`，`60 以下:E`。
11. 用 `switch` 实现简单计算器：输入两个数和运算符 `+ - * /`，输出结果。

## 第三阶段：循环

12. 计算 `1 + 2 + ... + 100`。
13. 计算 `1` 到 `100` 之间所有偶数的和。
14. 输出 `1` 到 `100` 之间所有能被 `3` 整除的数。
15. 输入一个整数 `n`，计算 `n!`。
16. 输出九九乘法表。
17. 判断一个数是否为质数。
18. 输出 `100` 以内所有质数。
19. 输入一个整数，反转它的数字，例如 `12345 -> 54321`。
20. 判断一个整数是否为回文数，例如 `121`。

## 第四阶段：数组

21. 定义一个整数数组，找出最大值、最小值、平均值。
22. 计算数组中所有元素的和。
23. 将数组倒序输出。
24. 在数组中查找某个元素是否存在。
25. 统计数组中偶数和奇数的个数。
26. 将一个数组复制到另一个数组中。
27. 对数组进行冒泡排序。
28. 合并两个有序数组。

## 第五阶段：字符串

29. 统计字符串中英文字母、数字、空格的个数。
30. 判断一个字符串是否为回文串。
31. 统计一个字符串中某个字符出现的次数。
32. 将字符串中的所有小写字母转为大写。
33. 把一句话中的单词顺序反转，例如 `I love Java -> Java love I`。

## 第六阶段：方法

34. 写一个方法，返回两个整数的最大值。
35. 写一个方法，判断一个数是否为质数。
36. 写一个方法，计算数组的平均值。
37. 写一个方法，实现字符串反转。
38. 写一个方法，求斐波那契数列第 `n` 项。

## 第七阶段：面向对象

39. 创建 `Student` 类，包含姓名、年龄、成绩属性，并提供输出信息的方法。
40. 为 `Student` 类添加构造方法和 `getter/setter`。
41. 创建 `Rectangle` 类，计算面积和周长。
42. 创建 `BankAccount` 类，实现存款、取款、查询余额。
43. 创建 `Person` 和 `Teacher` 类，练习继承。
44. 使用方法重载实现多个 `sum()` 方法。
45. 使用 `toString()` 输出对象信息。

## 第八阶段：综合基础

46. 写一个学生管理小程序：添加学生、查看学生、按姓名查找学生。
47. 写一个图书管理小程序：添加图书、借出图书、归还图书。
48. 用 `ArrayList` 存储若干整数，找出最大值并排序。
49. 使用 `HashMap` 统计一段字符串中每个字符出现的次数。
50. 模拟登录：用户名和密码正确才允许进入，最多输 `3` 次。

## 学习节奏

- 第 1 周：完成前 15 题，熟悉语法、判断、循环。
- 第 2 周：完成 16 到 33 题，强化数组和字符串。
- 第 3 周：完成 34 到 45 题，开始写方法和类。
- 第 4 周：完成 46 到 50 题，做综合练习。
